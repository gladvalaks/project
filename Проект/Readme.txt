MemeOtvet
Создатель - Майоров Илья

Сайт для людей, которые не шарят в мемах и нуждаются в помощи.
Пользователь может редатировать профиль, создавать вопросы, отвечать на вопросы, отмечать правльные ответы (только на свои вопросы). 
У пользователя есть баллы которыми он расплачивается за ответы на свой вопрос. И т.д.


flask, flask_sqlalchemy, shutil, os - библеотеки
Memeuser, Memecategory,Admins,Memequestion,Memeanswer - классы таблиц
